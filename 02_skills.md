# 02_skills.md — Ferramentas & Integrações

## Skills Instaladas

### Trello (trello/)
- Gerenciar 3 boards: Pós-Venda, Marketing, Projetos
- Leitura + escrita de cards, checklists, comentários

### Pipedrive (pipedrive/)
- CRM — deals, atividades, contatos, pipeline
- Leitura + escrita com confirmação

### Message Templates (message-templates/)
- Templates pré-aprovados para grupos de pós-venda
- Padronização de comunicação com clientes

### Market Monitoring (market-monitoring/)
- Monitoramento de mercado e concorrência
- Pesquisa via web_search

## Ferramentas Disponíveis

| Ferramenta | Uso | Escopo |
|------------|-----|--------|
| Trello API | Gestão de tarefas | 3 boards específicos |
| Pipedrive API | CRM | Deals, atividades, contatos |
| Google Drive | Documentação | Consulta |
| Google Sheets | Planilhas | Consulta |
| Brave Search | Pesquisa de mercado | Concorrência, trends |
| Himalaya | Email (futuro) | IMAP/SMTP |
| WhatsApp Gateway | Mensagens | Grupos de pós-venda |
