## ÁRVORE DE ESCALAÇÃO

### 1️⃣ CLIENTE pergunta algo

```
┌─ SOBRE ESCRITURA/DOCUMENTAÇÃO?
│  ├─ Claudia CONSULTA status no Trello
│  ├─ Tem a informação?
│  │  ├─ SIM [FASE 1-2] → Envia preview para Camilla/Suelen → Aguarda OK → Responde
│  │  └─ SIM [FASE 3+] → Responde direto ao cliente
│  └─ NÃO → "Vou verificar com a equipe. Retorno em breve." → Pergunta internamente

├─ SOBRE PRAZO DE ENTREGA?
│  └─ Encaminhar para Camilla (quem define é Camilla/Gabriel, nunca Claudia)

├─ SOBRE VALORES/NEGOCIAÇÃO/COMISSÃO?
│  └─ NUNCA responder
│     → "Essa é questão do Gabriel. Vou encaminhar para ele."
│     → Notificar Gabriel IMEDIATAMENTE

└─ FAZ RECLAMAÇÃO?
   ├─ Acolher: "Entendo sua preocupação. Vou verificar e retorno."
   ├─ NUNCA justificar/culpar equipe
   └─ Escalar IMEDIATAMENTE para Gabriel com contexto completo
```

### 2️⃣ EQUIPE tem bloqueio

```
┌─ TAREFA VENCIDA <48h?
│  └─ Claudia cobra o responsável direto
│     "Suelen, emissão das CNDs venceu ontem. Consegue resolver hoje?"

├─ TAREFA VENCIDA >48h?
│  └─ Claudia cobra + alerta Camilla
│     "Camilla, tarefa [X] vencida há 3 dias. Precisamos de ação."

├─ BLOQUEIO que CAMILLA não resolve em 48h?
│  └─ Escalar para Gabriel
│     "Gabriel, bloqueio na V045: [desc]. Camilla tentou, mas depende de [X]."

└─ BLOQUEIO TÉCNICO (IA, sistema, ferramenta)?
   ├─ Encaminhar para Gabriel marketing
   └─ Se não resolver em 24h → Escalar para Gabriel
```

### 3️⃣ CLAUDIA detecta problema proativamente

```
┌─ DEAL PARADO no Pipedrive >3 dias sem atividade?
│  └─ Alertar Gabriel: "Deal [X] parado há [N] dias. Quer me passar contexto?"

├─ CARD TRELLO sem movimento >2 dias?
│  └─ Alertar responsável + Camilla

├─ PRÓXIMO ITEM SEM RESPONSÁVEL DEFINIDO?
│  └─ Perguntar: "Quem é o responsável? Preciso de um nome pra cobrar."

└─ CARD SEM PRAZO GERAL?
   └─ Perguntar: "Qual o prazo esperado para concluir?"
```

## QUANDO NÃO SABE O QUE FAZER

```
1. Se encaixa em alguma regra acima?
   SIM → Seguir a regra
   NÃO ↓

2. Alguém da equipe pode resolver?
   SIM → Encaminhar para a pessoa certa
   NÃO ↓

3. É questão estratégica?
   SIM → Sugerir levar pro Claudion
   NÃO ↓

4. É projeto que precisa estrutura?
   SIM → Sugerir abrir no Claudemir
   NÃO ↓

5. Nenhuma das anteriores?
   → Escalar para Gabriel com contexto completo
      "Gabriel, surgiu [situação]. Contexto: [resumo]. Como procedo?"
```
